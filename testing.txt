  1. Detection Accuracy (Model Quality)                 
                                       
  ┌───────────────────────────────┬───────────────────────────────────────────────┬────────────────────┬────────────────────────────────────────────────────┐
  │            Metric             │               What it measures                │     Good value     │                     Limitation                     │

  │ mAP@50                        │ Detection accuracy at IoU ≥ 0.50              │ >0.85              │ Loose overlap threshold — hides box quality issues │               
  ├───────────────────────────────┼───────────────────────────────────────────────┼────────────────────┼────────────────────────────────────────────────────┤
  │ mAP@50-95                     │ Average over IoU 0.50–0.95 (10 thresholds)    │ >0.65              │ More honest; harder to game                        │               
  ├───────────────────────────────┼───────────────────────────────────────────────┼────────────────────┼────────────────────────────────────────────────────┤               
  │ Precision                     │ Of all detections made, how many were correct │ >0.90              │ High precision, low recall = misses objects        │               
  ├───────────────────────────────┼───────────────────────────────────────────────┼────────────────────┼────────────────────────────────────────────────────┤               
  │ Recall                        │ Of all real objects, how many were caught     │ >0.90              │ High recall, low precision = many false alarms     │
  ├───────────────────────────────┼───────────────────────────────────────────────┼────────────────────┼────────────────────────────────────────────────────┤               
  │ F1 Score                      │ Harmonic mean of precision and recall         │ >0.88              │ Single balanced view                               │
  ├───────────────────────────────┼───────────────────────────────────────────────┼────────────────────┼────────────────────────────────────────────────────┤               
  │ Per-class AP                  │ AP broken down per class (carton, polybag)    │ —                  │ Reveals class imbalance effects                    │
  ├───────────────────────────────┼───────────────────────────────────────────────┼────────────────────┼────────────────────────────────────────────────────┤

  10. Domain-Specific Limitations of This System        
                                                                                                                                                                            
  ┌──────────────────────────────┬────────────────────────────────────────────────────┬─────────────────────────────────────────────────────────────┐
  │          Limitation          │                     Root cause                     │                     Observable symptom                      │                       
  ├──────────────────────────────┼────────────────────────────────────────────────────┼─────────────────────────────────────────────────────────────┤
  │ Polybag transparency         │ Polybags reflect IR and specular light             │ Low mask IoU, low recall for polybags                       │
  ├──────────────────────────────┼────────────────────────────────────────────────────┼─────────────────────────────────────────────────────────────┤
  │ High-density crossing        │ Multiple objects crossing simultaneously           │ ID swaps → double-counts                                    │                       
  ├──────────────────────────────┼────────────────────────────────────────────────────┼─────────────────────────────────────────────────────────────┤                       
  │ Fast conveyor speed          │ Object appears in <3 frames                        │ Track too short to cross the line with bottom-center anchor │                       
  ├──────────────────────────────┼────────────────────────────────────────────────────┼─────────────────────────────────────────────────────────────┤                       
  │ Partial occlusion            │ One parcel behind another                          │ Track interrupted → ID switch on reappear                   │
  ├──────────────────────────────┼────────────────────────────────────────────────────┼─────────────────────────────────────────────────────────────┤                       
  │ Lighting variation           │ Factory shift changes, lamp flicker                │ Confidence scores drop unpredictably                        │
  ├──────────────────────────────┼────────────────────────────────────────────────────┼─────────────────────────────────────────────────────────────┤                       
  │ Model class imbalance        │ If cartons >> polybags in training data            │ Per-class AP diverges significantly                         │
  ├──────────────────────────────┼────────────────────────────────────────────────────┼─────────────────────────────────────────────────────────────┤                       
  │ Resolution vs speed tradeoff │ Higher res = better small object detection, slower │ RF-DETR at 640px: ~40ms; at 448px: ~25ms                    │
  ├──────────────────────────────┼────────────────────────────────────────────────────┼─────────────────────────────────────────────────────────────┤                       
  │ Line position sensitivity    │ Line at wrong X percent misses fast objects        │ Objects may jump over the line between frames               │
  ├──────────────────────────────┼────────────────────────────────────────────────────┼─────────────────────────────────────────────────────────────┤                       
  │ Single counting line         │ Only one line, one direction                       │ Can't distinguish objects entering vs leaving the frame     │
  └──────────────────────────────┴────────────────────────────────────────────────────┴─────────────────────────────────────────────────────────────┘
